Projects

Minesweeper

The classic game Minesweeper.
A tile identifies if the tile to its left and right and up and down has a mine and increments the displayed clicked tile number. You win if all mines are flagged and all other tiles are clicked.
Uses React and Vite.

Counters

Counters is a counting app using React Native, currently used to aid me in my current work. It has several counters that are saved to local storage.
Uses React Native, Expo, Zustand

Flashcards

Flashcards was created from an idea to have digital language cards. Specifically English and German. Currently you can have a database of German words which ‘flip’ when clicked to the English word. A gamified version is a work in progress.
Flashcards connects through an API created by me (dbApi) to a Mongo database.
Uses React, Webpack, Babel and ChakraUI.

dbApi

The API for connecting the flashcards project to the database storing the words. Uses Node, Express and Mongoose.
